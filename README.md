OsmoIRSensor
IR Water &amp; Fluid detection Sensor based on ATTINY45. 
- KiCad Shematic & PCB included under hardware
- Firmware for ATTINY45 included under software

The Sensor was designed for Osmosi waterlevel control. 
It is designed to be full encapsulated in a EPOXY case. 
The hole design an maufacturing process as well as the 
technical backgrounds for the IR detection 
is described under
